How It Works
Somnia Agents are off-chain validators that execute AI tasks (API calls, LLM inference, web scraping) and return results on-chain via the platform contract at 0x037Bb9C718F3f7fe5eCBDB0b600D607b52706776.
Each request sends a payload + STT deposit to the platform. Validators execute the agent, reach consensus, and the platform calls back the contract with the result. The deposit is refunded minus execution costs.
PriceOracle uses the JSON API Request agent to fetch from CoinGecko. SentimentAnalyzer classifies crypto market sentiment (bullish/bearish/neutral) via LLM. HealthTips uses the LLM Inference agent (Qwen3-30B) with chain-of-thought for evidence-based health advice. DrugInfoV2 uses the LLM agent for AI-generated drug information as a replacement for the non-functional Parse Website agent.
JSON API: 0.12 STT (0.03 deposit + 3 validators), LLM: 0.24 STT, Parse Website: 0.33 STT. Deposits are refunded on success. We have 0.07 STT remaining — refill from the faucet to invoke new requests.
